7.What are attributes?

ANS:Attributes define additional information of the element such as width and height of an image. Attributes are always specified in the start tag (or opening tag) and usually consists of name/value pairs like name="value". Attribute values should always be enclosed in quotation marks. Also, some attributes are required for certain elements. For instance, an img tag must contain a src and alt attributes. we can add multiple atributes to particular tag
